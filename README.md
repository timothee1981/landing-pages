# Music Landing Pages

> 6 High-Converting Landing Page Templates for Musicians

![Tailwind](https://img.shields.io/badge/Tailwind-3.4-blue)

## Quick Start

```bash
# Install
npm install

# Develop
npm run dev

# Build
npm run build
```
